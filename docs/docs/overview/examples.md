# Example applications

There are several example applications for various Invirt capabilities and use cases
in the [examples](https://github.com/resoluteworks/invirt/tree/main/examples) directory.
All of these are designed as applications that can be built and run.

To run any of these examples, use the `run` script inside the `examples` directory.
```shell
cd examples
./run quickstart
```

Available examples:
* [quickstart](https://github.com/resoluteworks/invirt/tree/main/examples/quickstart) - minimal Invirt application
* [form-basics](https://github.com/resoluteworks/invirt/tree/main/examples/form-basics) - HTML form binding to typed objects
* [form-validation](https://github.com/resoluteworks/invirt/tree/main/examples/form-validation) - server-side form validation
* [data-querying](https://github.com/resoluteworks/invirt/tree/main/examples/data-querying) - filters, sort and pagination from query parameters
* [security-authentication](https://github.com/resoluteworks/invirt/tree/main/examples/security-authentication) - cookie-based authentication
* [static-assets](https://github.com/resoluteworks/invirt/tree/main/examples/static-assets) - serving and versioning static assets
